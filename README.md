

# 📈 株価テクニカル分析ツール（Python / Streamlit）

## 📌 概要
本ツールは、株価データを取得し、テクニカル指標（RSI、MACD、移動平均線など）を用いて分析を行うWebアプリケーションです。

ユーザーは銘柄コードを入力するだけで、リアルタイムにチャートと売買判断の参考情報を確認できます。

---

## 🎯 主な機能
- 株価データの自動取得（Yahoo Finance）
- テクニカル指標の計算（RSI / MACD / 移動平均線）
- 出来高分析
- トレンド分析（短期・中期）
- サポート・レジスタンス分析
- インタラクティブなチャート表示

---

## 🛠 使用技術
- Python
- Streamlit
- yfinance
- pandas
- matplotlib
